# IFT 3395 – Kaggle 2

Créer un environnement, préférablement `conda`, et installer les dépendances nécessaires de `requirements.txt`.

Pour utiliser avec `jupyter notebook`, n'oubliez pas d'ajouter l'environnement en tant que kernel Jupyter :
```
python -m ipykernel install --user --name <name_of_env> --display-name "Python (<name_of_env>)"
```
